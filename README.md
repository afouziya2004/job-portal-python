## Project Overview
A RESTful Job Portal backend built using Python and Flask.
Supports CRUD operations with persistent file storage and validation.

## Tech Stack
- Python
- Flask
- JSON
- File-based persistence

## Features
- Add job (API)
- List jobs (API)
- Delete job (API)
- Input validation
- Error handling
- Persistent storage

## API Endpoints
GET /jobs  
POST /jobs  
DELETE /jobs/<id>

## How to Run
pip install flask  
python app.py
